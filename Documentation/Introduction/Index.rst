What does it do?
================

Taskhub is a TYPO3 extension that provides task management functionality. It allows users to create, edit, delete, and manage tasks, including assigning tasks to frontend users and categorizing them.

Features

- Create, edit, and delete tasks
- Assign tasks to frontend users
- Categorize tasks using TYPO3's system categories
- Mark tasks as done/undone
- Set due dates and reminder dates for tasks
- List view with filtering options
- Detailed view of individual tasks

Usage:
Frontend

Navigate to the page where you've added the Task List plugin.
Use the filter options to find specific tasks.
Click on a task to view its details.
Use the provided buttons to mark tasks as done/undone or to edit/delete tasks.

Backend

Create a new task by navigating to List view and selecting your storage folder.
Click on "Create new record" and select "Task".
Fill in the task details, including title, description, due date, etc.
Save the task.
